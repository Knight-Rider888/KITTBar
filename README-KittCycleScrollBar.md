# KittCycleScrollBar
上下滚动栏


- xml属性

```
<declare-styleable name="KittCycleScrollBar">

    <!--自动滚动间隔，默认间隔2000毫秒-->
    <attr name="bar_auto_scroll_interval" format="integer" />

</declare-styleable>
```


- addItemView() 添加待滚动的view

- clearAllViews()  清空所有view

- onResume()  生命周期进行调用

- onPause()  生命周期进行调用
